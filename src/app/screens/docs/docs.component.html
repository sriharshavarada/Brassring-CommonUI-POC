Consumer Docs

This section explains how any team can consume `br-grid`, `br-date`, and `br-modal` in their own app.

Quick Start

  1. Import the fa√ßade components from the common layer only.
  2. Pass library-agnostic JSON (`BrGridConfig`, `BrDateConfig`, `BrModalConfig`).
  3. Handle emitted events in your feature component or service.
  4. Switch rendering mode from Playground or mode config without touching consumer screens.

Usage Example

<br-grid [config]="gridConfig" (action)="onGridAction($event)"></br-grid>
<br-date [config]="dateConfig" (dateChange)="onDateChange($event)"></br-date>
<br-modal [config]="modalConfig" (action)="onModalAction($event)"></br-modal>

Event Wiring Pattern

onGridAction(event: BrGridActionEvent) {
  if (event.actionId === 'save') {
    this.api.saveSelection(event.selectedRows).subscribe();
  }
}

onModalAction(event: BrModalActionEvent) {
  if (event.actionId === 'confirm') {
    this.api.confirm(event.fieldValues).subscribe();
  }
}
